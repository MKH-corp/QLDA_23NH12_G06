# Work & KPI Frontend - Sprint 1

React + Vite frontend for Work & KPI Management.

## Features
- Kanban board with 4 columns: Todo, Doing, Done, Blocked
- Task card shows title, priority, due date
- Create task form
- Edit task form
- Delete task
- Call real FastAPI backend APIs

## Local run
1. Copy `.env.example` to `.env`
2. Install dependencies with `npm install`
3. Start dev server with `npm run dev`
4. Ensure backend is running at `VITE_API_BASE_URL`
